# WLED Expressive 🎨💡

A modern, Material 3-focused remote control app for WLED devices.

## Features ✨

- **Material 3 Expressive Design**: Beautiful, modern UI with "Expressive" sliders that match the latest Android system designs.
- **Dynamic Color Support**: Seamless integration with Android 12+ Material You dynamic coloring.
- **Theme Customization**: Toggle between Light, Dark, and System modes. Manual seed color selection when dynamic color is disabled.
- **Master Control**: Control all online WLED devices at once (global brightness, power off).
- **Auto-Discovery**: Automatically find WLED devices on your local network using mDNS.
- **Device Management**: Add, edit, and identify devices with ease.
- **Full Control**: Deep control over colors, effects, palettes, and presets.
